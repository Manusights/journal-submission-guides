# How to Handle Null Results in Submission

Null results are publishable when design is strong.

State:
- Why the question matters
- Why the null result is informative
- Limits and context clearly


## Practical sequence
1. Confirm target journal fit
2. Run claim-to-evidence map
3. Check references tied to core claims
4. Final pass on methods and limitations
