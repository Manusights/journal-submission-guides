# How to Map Claims to Figures Before Submit

Create a two-column table:
- Claim
- Supporting figure/panel

Remove or soften claims without direct visual support.


## Practical sequence
1. Confirm target journal fit
2. Run claim-to-evidence map
3. Check references tied to core claims
4. Final pass on methods and limitations
