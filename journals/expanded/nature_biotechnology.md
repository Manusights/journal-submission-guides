# Nature Biotechnology Submission Guide

## Journal profile
- Primary audience: Biotechnology researchers, platform builders, translational teams
- Editorial focus: Technical innovation plus convincing performance and impact
- Best use case: Submit when method novelty and benchmark strength are both clear.

## What editors usually screen first
1. Novelty of platform or method
2. Fair and transparent benchmarking
3. Reproducibility detail for critical steps
4. Real-world relevance of performance gains

## Practical pre-submission checks
- Make a benchmark table with comparator methods
- State dataset splits and evaluation logic clearly
- Include reproducibility details for code, protocol, and data
- Keep claim strength matched to benchmark quality

## Common avoidable rejection patterns
- Platform novelty is unclear versus existing methods
- Benchmarking setup is incomplete or not fair
- Reproducibility details are missing for key pipeline steps

## Submission package checklist
- One-page benchmark summary
- Reproducibility notes for methods section
- Cover letter that states specific technical advance
