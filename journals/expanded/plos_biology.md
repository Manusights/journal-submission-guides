# PLOS Biology Submission Guide

## Journal profile
- Primary audience: Broad biology audience
- Editorial focus: Strong biology with clear significance and open science practice
- Best use case: Submit when findings are both important and transparently reported.

## What editors usually screen first
1. Significance and fit for broad biology readership
2. Reporting quality and transparency
3. Data and source organization quality
4. Claim strength versus evidence

## Practical pre-submission checks
- Run an open-science readiness check
- Align claims between abstract, results, and discussion
- Organize source data files by figure and panel
- Verify references attached to key conclusions

## Common avoidable rejection patterns
- Broad claim without sufficient cross-context evidence
- Reporting quality uneven across sections
- Source data organization difficult for reviewers

## Submission package checklist
- Open data/readability pass
- Cross-section claim consistency check
- Structured source data map
