# How to Pick Primary and Backup Journals

Choose one primary and two backups before first submission.

Rule:
- Primary: best conceptual fit
- Backup 1: realistic scope and audience fit
- Backup 2: faster route with solid relevance


## Practical sequence
1. Confirm target journal fit
2. Run claim-to-evidence map
3. Check references tied to core claims
4. Final pass on methods and limitations
