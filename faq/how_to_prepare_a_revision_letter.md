# How to Prepare a Revision Letter

Use point-by-point format with quoted reviewer comments.

Include exact manuscript changes by page and line. Keep disagreement evidence-based.


## Practical sequence
1. Confirm target journal fit
2. Run claim-to-evidence map
3. Check references tied to core claims
4. Final pass on methods and limitations
