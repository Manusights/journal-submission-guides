# How to Write a Cover Letter Editors Actually Read

Keep it short:
- Problem
- Main result with one number
- Why this journal audience should care
- Compliance statements


## Practical sequence
1. Confirm target journal fit
2. Run claim-to-evidence map
3. Check references tied to core claims
4. Final pass on methods and limitations
