# How to Check Reference Integrity Quickly

Sample high-impact references first.

Verify:
- DOI resolves
- Title/year match
- Citation supports nearby claim


## Practical sequence
1. Confirm target journal fit
2. Run claim-to-evidence map
3. Check references tied to core claims
4. Final pass on methods and limitations
