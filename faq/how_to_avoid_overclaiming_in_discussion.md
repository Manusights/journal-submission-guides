# How to Avoid Overclaiming in Discussion

Use one boundary sentence for each major conclusion.

Do not present exploratory findings as definitive outcomes.


## Practical sequence
1. Confirm target journal fit
2. Run claim-to-evidence map
3. Check references tied to core claims
4. Final pass on methods and limitations
